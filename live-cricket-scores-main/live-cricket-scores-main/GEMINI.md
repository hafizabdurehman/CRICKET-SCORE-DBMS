# Live Cricket Scores Flask Application

This is a Python Flask application that displays live cricket scores from an RSS feed.

## Setup Instructions

1.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    ```

2.  **Activate the virtual environment:**
    *   **Windows:**
        ```bash
        venv\Scripts\activate
        ```
    *   **macOS/Linux:**
        ```bash
        source venv/bin/activate
        ```

3.  **Install the required packages:**
    ```bash
    pip install Flask feedparser requests
    ```

## How to Run the Application

1.  **Run the Flask application:**
    ```bash
    python app.py
    ```

2.  **Open your web browser and go to the following URL:**
    [http://127.0.0.1:5000/](http://127.0.0.1:5000/)
